# Agitation Detection Using Wearable Biosignals
Lightweight deep learning pipeline for detecting agitation in dementia care

## Overview
This project aims to explore agitation detection using wearable sensor data (including: Heartrate, Temperature, Electrodermal Activity, and Accelerometer).

## Objectives
- Build a biosignal preprocessing pipeline
- Train deep learning models for agitation detection
- Explore edge-optimised architectures
